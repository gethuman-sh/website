# human — The Dark Software Factory for Claude Code

> {{ .Site.Params.description }}

human is a CLI harness for running a "dark software factory" on a codebase: AI agents pick up tickets and write, test, and ship code while humans stay on intent, policy, and outcome review. It connects issue trackers and tools (Jira, GitHub, Linear, Notion, Figma, Amplitude, Sentry) with roughly 95% fewer tokens than MCP servers, and ships a secure devcontainer, Chrome Bridge, and lifecycle skills.
